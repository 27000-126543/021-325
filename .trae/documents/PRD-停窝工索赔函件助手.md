
## 1. 产品概述

AI 辅助的停窝工索赔函件生成工具，面向施工企业法务、合同管理员和项目商务人员，帮助非法律背景人员规范撰写索赔函件，减少函件口径随意和漏报时限风险。

- 核心价值：将事实、依据、诉求结构化录入，自动生成规范函件并标注缺失证据
- 目标用户：施工企业法务、合同管理员、项目商务人员

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（本地工具） | 使用全部函件生成功能 |

### 2.2 功能模块
1. **首页/索赔类型选择页**：三种索赔类型卡片式选择，功能介绍
2. **信息填写页**：结构化表单录入合同条款、事件经过、天数、费用、证据
3. **函件预览页**：生成函件初稿 + 缺失证据醒目标注 + 函件类型切换
4. **函件修改页**：措辞强硬程度滑块调整（商务协商 ↔ 合同维权），实时预览

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 索赔类型选择 | 卡片式展示三种索赔类型（业主原因停工、设计变更窝工、甲供材料延误），点击进入对应表单 |
| 首页 | 工具说明 | 简要介绍工具用途和使用流程 |
| 信息填写页 | 索赔类型展示 | 顶部显示当前选择的索赔类型，可返回重新选择 |
| 信息填写页 | 基础信息表单 | 合同条款编号、事件经过（多行文本）、已确认天数、已发生费用 |
| 信息填写页 | 证据清单 | 多选框列出常见证据类型，支持自定义补充 |
| 信息填写页 | 生成按钮 | 校验必填项后生成函件 |
| 函件预览页 | 函件类型切换 | 索赔意向通知 / 费用索赔报告 / 催办函 三种类型切换 |
| 函件预览页 | 函件内容展示 | 格式化展示生成的函件正文 |
| 函件预览页 | 缺失证据标注 | 醒目红色标注区域，列出缺少的事实依据 |
| 函件预览页 | 操作按钮 | 复制内容、导出文本、进入修改页 |
| 函件修改页 | 措辞强度滑块 | 从"偏商务协商"到"偏合同维权"的连续调整 |
| 函件修改页 | 实时预览 | 调整滑块后函件措辞实时更新 |
| 函件修改页 | 段落微调 | 各段落可单独编辑 |

## 3. 核心流程

用户从首页选择索赔类型 → 填写结构化表单（合同条款、事件经过、天数费用、证据）→ 系统生成函件初稿并标注缺失证据 → 用户可切换函件类型（意向通知/索赔报告/催办函）→ 进入修改页调整措辞强硬程度 → 复制或导出最终函件

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深青色（#0F766E）作为主色，体现专业、稳重；暖橙色（#F97316）作为强调色，用于缺失证据警示
- **辅助色**：浅灰背景（#F8FAFC），白色卡片，中性灰文字层级
- **按钮风格**：圆角矩形（8px），主按钮深青色填充，悬停加深；次级按钮描边样式
- **字体**："Noto Serif SC" 作为标题字体（法律文书感），"Noto Sans SC" 作为正文字体
- **布局风格**：左右分栏（左侧表单/右侧预览），卡片式容器，清晰的视觉层级
- **图标风格**：Lucide 线性图标，统一 20px 尺寸

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|----------|
| 首页 | 索赔类型卡片 | 三列等宽卡片，图标 + 标题 + 描述，悬停上浮阴影 |
| 信息填写页 | 表单区域 | 分组卡片，标签左对齐，输入框统一高度，必填红星标记 |
| 函件预览页 | 缺失证据警示 | 橙色背景边框 + 警告图标 + 红色加粗文字列表 |
| 函件预览页 | 函件正文 | 仿 A4 纸张效果（白色 + 阴影 + 内边距），衬线字体 |
| 函件修改页 | 措辞滑块 | 轨道渐变（蓝色 → 红色），两端标签说明，实时数值显示 |

### 4.3 响应式
- 桌面端优先（≥1280px）：左右分栏布局
- 平板端（768-1279px）：上下堆叠，表单和预览各占全宽
- 移动端（<768px）：单列布局，简化间距，触摸友好按钮
